title: Работа с файлами
---

В Node.js для работы с файлами можно использовать модуль fs. Этот модуль предоставляет множество функций для работы с файлами на диске, таких как чтение, запись и удаление файлов.

Вот пример кода, который открывает файл “input.txt” для чтения и выводит содержимое файла в консоль:

const fs = require(‘fs’);

// открываем файл для чтения
const data = fs.readFileSync(‘input.txt’, ‘utf8’);

console.log(data);

В данном примере мы импортируем модуль fs и открываем файл “input.txt” с помощью функции readFileSync(). Затем мы передаем путь к файлу и кодировку, в которой файл должен быть прочитан, в функцию readFileSync().

Если мы хотим записать данные в файл, то мы можем использовать функцию writeFileSync().
